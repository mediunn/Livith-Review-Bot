# knowledge/ — PRD 스냅샷

이 디렉터리에는 현재 스프린트의 **PRD·요구사항(FR) 스냅샷**(`<스프린트>.md`)이 들어간다.
**비공개 제품 데이터라 `.gitignore`로 제외**되어 있다 (이 README만 커밋됨).

## 스냅샷이란

실시간 Notion 조회는 비결정적(플랜 제한·레이트리밋)이라, PRD+FR 전체를 **한 번 추출해 고정**한 마크다운.
이 스냅샷이 Hermes 프로파일의 `SOUL.md`(항상 로드되는 시스템 프롬프트)에 내장되어, 봇이 런타임 검색 없이 결정적으로 답한다.

## 생성 / 갱신

1. 통합 토큰 Notion MCP(`notion-api`)로 PRD 페이지 + 요구사항 DB를 추출
   (레이트리밋 회피를 위해 DB별로 간격을 두고 호출)
2. FR 목록을 구조화해 `<스프린트>.md`로 저장
3. 같은 내용을 프로파일 SOUL의 「지식」 섹션에 동기화

PRD 변경 시 재추출 → 동기화. `hermes cron`으로 매일 자동 갱신(검증 통과 시에만 적용, 실패 시 기존 보존).
